## Development

```console
$ git clone https://github.com/arnaudjuracek/www-arnaudjuracek
$ cd www-arnaudjuracek
$ yarn install
$ yarn start
```

## Deployment

Deployment is done automatically via a [Github action](.github/workflows/main.yml). Create a new release by running:
```console
$ yarn version
```
